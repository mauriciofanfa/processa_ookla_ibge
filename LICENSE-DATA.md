# Licença dos dados derivados

Os dados processados gerados por `processa_ookla_ibge.py` incorporam dados da Ookla
Global Performance Maps, disponibilizados sob a licença Creative Commons
Attribution-NonCommercial-ShareAlike 4.0 International (`CC BY-NC-SA 4.0`).

Por esse motivo, os dados derivados devem ser publicados e redistribuídos sob a mesma
licença, preservando as condições de atribuição, uso não comercial e compartilhamento pela
mesma licença.

Texto da licença:

https://creativecommons.org/licenses/by-nc-sa/4.0/

Ao publicar os dados derivados, recomenda-se indicar a fonte original: Speedtest by Ookla
Global Fixed and Mobile Network Performance Maps.

Esta nota trata especificamente dos dados derivados. A licença geral do repositório,
incluindo código-fonte, documentação, arquivos de apoio, metadados e demais produtos
autorais, está registrada no arquivo `LICENSE`.
